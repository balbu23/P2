Protocolo TCP/IP

Protocolo de transporte que se encarga de garantizar la entrega de los datos en la comunicación en red.

Este termino (TCP/IP) se refiere especificamente a dos protocolos:
TCP (Transmisión Control Protocol) y IP (Internet Protocol).















Importancia del protocolo TCP/IP radica en su capacidad para permitir la
comunicación entre dispositivos y sistemas de diferentes fabricantes y
Sistemas operativos.





¿SUS PAQUETES DE DATOS SON
PRIVADOS A TRÁVES DE TCP/IP?










Los paquetes de datos enviados a través de TCP/IP no son privados, lo que
significa que se pueden ver o interceptar.

CARACTERISTICAS



El término TCP/IP se refiere a un cojunto de
protocolos de comunicación de red
que permiten la comunicación entre dispositivos y
sistemas










Se encarga de dividir los datos en paquetes y asignarles un
número
de secuencia para garantizar su entrega en el orden correcto.












El protocolo TCP/IP se compone de cinco capas, cada una con sus
propias
funciones y características.

















1-CAPA DE APLICACIÓN

2-CAPA DE TRANSPORTE

3-CAPA DE RED

4-CAPA DE INTERFAZ DE RED

5-CAPA DE HARWARE







DESVENTAJAS:

1.Complejidad: Puede requerir una gran cantidad de
configuración y
mantenimiento.


2.Rendimiento:Puede ser lento en comparación con
otros protocolos,
Especialmente en redes de alta velocidad.


3.Dependencia de la conexión: Requiere una conexión
establecida entre la
fuente y el destino para funcionar

Ventajas/Desventajas

Ventajas:

1.Fácil implementación: Accesible para una amplia
variedad de usuarios y
aplicaciones.

2.Escalabilidad: Puede manejar grandes cantidades de
tráfico de red sin
afectar su rendimiento.

3.Flexibilidad: Capaz de adaptarse a diferentes entornos y
aplicaciones.
